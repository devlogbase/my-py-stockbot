# 🤖 MyPyStockbot
<br/>

## 📢 Introduction

- 주식 관련 정보를 알려주는 텔레그램 챗봇 프로그램입니다.
- 주가지수, 환율정보 및 ETF 가격 체크 등의 기능을 지원합니다.
- 데이터 베이스를 이용하여 사용자별 알람 On/Off 기능을 제공합니다.
